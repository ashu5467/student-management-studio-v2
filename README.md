Installation Guide

Frontend

commands to run for Frontend libraries

cd /Desktop/Student\ Studio\ V2/Student\ Studio/FrontEnd/app

npm install 
npm install react react-dom xlsx 
npm install tailwindcss 
npm install file-saver 
npm install axios

Library details
1.react & react-dom: Core libraries for building the React application.
2.xlsx: Library for handling Excel file operations.
3.tailwindcss: Utility-first CSS framework for styling.
4.file-saver: Library to save files (like PDFs or Excel files) on the client side.
5.axios: HTTP client for making API requests to the backend.



Backend

commands to run for Backend libraries

cd /Desktop/Student\ Studio\ V2/Student\ Studio/BackEnd



npm install
npm install express cors exceljs psd pdf-lib xlsx sharp
npm install express multer fs-extra xlsx cors socket.io sharp body-parser psd


Library details
1.express: Web framework for Node.js.
2.cors: Middleware for handling cross-origin resource sharing.
3.exceljs: Library for reading and writing Excel files.
4.psd: Library for handling PSD (Photoshop) files.
5.pdf-lib: Library for working with PDFs.
6.xlsx: Library for parsing and writing Excel files.
7.sharp: High-performance image processing library.
8.multer: Middleware for handling file uploads.
9.fs-extra: Adds extra filesystem methods to Node.js.
10.socket.io: Library for real-time bidirectional communication.
11.body-parser: Middleware for parsing incoming request bodies.
